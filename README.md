Getting_Cleaning_data
=====================
run_analysis.R          R code that can obtain the tidy data.

data should be extract to the same directory as this code.

data:
train           training set
test            test set
data_complete   combination of training and test sets

data_filtered   mean and SD for each measurement

data_named      data with descriptive activity names

result          tidy data set with the average of each variable for each activity and each subject
